# Searching-Algorithms
Implementation of searching algorithms like Linear Search and Binary Search in C++

## Algorithms Included-
-Linear Search
-Binary Search

## Time Complexity-
-Linear Search--
    Best-O(1)
    Worst-O(n)
    Average-O(n)
-Binary Search--
    Best-O(1)
    Worst-O(logn)
    Average-O(logn)
